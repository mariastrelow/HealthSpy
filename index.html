Como Jogar

Insira de 4 a 8 jogadores.

Um jogador será o espião.
Os outros receberão um tema secreto.

Cada jogador faz uma pergunta para outro.
O espião deve tentar descobrir o tema.

Todos votam em quem acham que é o espião.

No final, o espião pode tentar adivinhar o tema para vencer.
